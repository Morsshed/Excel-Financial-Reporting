## Profit & Loss Analysis
![Profit & Loss Example](https://raw.githubusercontent.com/Morsshed/Excel-Financial-Reporting/main/ExcelImages/Profit&Loss.png)

###### Net Profit
                        Revenue - COGS
### Description
| Section                       | Description                                                                |
| ----------------------------- | -------------------------------------------------------------------------- |
| **Revenue (Sales)**           | Total income earned from goods or services sold.                           |
| **Cost of Goods Sold (COGS)** | Direct costs tied to producing goods or services (materials, labor, etc.). |
| **Gross Profit**              | Revenue minus COGS.                                                        |
| **Operating Expenses**        | Indirect costs like rent, salaries, utilities, marketing.                  |
| **Operating Income**          | Gross Profit minus Operating Expenses.                                     |
| **Net Profit (or Loss)**      | Final bottom-line figure after all expenses and taxes.                     |
